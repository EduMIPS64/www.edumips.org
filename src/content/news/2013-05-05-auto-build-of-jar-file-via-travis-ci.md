title: "Auto-build of JAR file via Travis CI"
date: 2013-05-05
kind: note
---

We added to our [Travis CI](https://travis-ci.org/lupino3/edumips64) [configuration](https://github.com/lupino3/edumips64/blob/master/.travis.yml) the [automatic build of JAR files](https://github.com/lupino3/edumips64/blob/master/utils/update-gh-pages.sh), so if you want to try the bleeding edge you can do it without having to compile it. The URL pointing to the always up-to-date JAR is [http://lupino3.github.io/edumips64/edumips64-latest.jar](http://lupino3.github.io/edumips64/edumips64-latest.jar). Be aware that it might be unstable.
