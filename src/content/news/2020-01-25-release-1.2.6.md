title: "Release 1.2.6 (codename: Phlegmatic)"
date: 2020-01-25
---

EduMIPS64 1.2.6 is out! This release fixes some bugs related to memory handling (OOM), increases the amount of memory available to programs, adds a better way to report bugs and contains the usual miscellaneous bugfixes and improvements related to the dev side of EduMIPS64 (e.g., migrating back to Gradle, using Azure Pipelines for CI, etc)) Here is the link to the release on [GitHub](https://github.com/lupino3/edumips64/releases/tag/v1.2.6).
